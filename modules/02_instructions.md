# Lição 2. Instructions

> Fonte: Dynatrace University — Associate Certification Learning Path | Lição 2 de 12

---

Este learning path é a chave para o sucesso na certificação. Recomenda-se seguir as lições em ordem para melhor preparação.

---

## Pré-requisitos

Antes de iniciar o Dynatrace Associate Learning Path e agendar o exame, garanta que você atende aos seguintes pré-requisitos:
| Área | Conhecimento Esperado |
|---|---|
| **Arquitetura** | Familiaridade com arquiteturas Service Oriented Application (SOA) |
| **Application Servers** | WebSphere, WebLogic, JBoss, Tomcat, IIS, etc. |
| **Aplicações** | Estrutura de apps mobile e web e sua arquitetura de hospedagem (bancos de dados, redes, processos) |
| **Performance** | Familiaridade geral com métricas de performance de aplicações |
| **Sistemas Operacionais** | Windows, UNIX, VMware, etc. |
| **Cloud & New Stack** | Azure, Docker, Kubernetes, Cloud Foundry, IBM Cloud, OpenStack, OpenShift, etc. |
| **Dynatrace** | 6 meses de experiência com a plataforma *(recomendado)* ||
---

## Como Aproveitar o Learning Path

1. **Percorrer todas as seções** do Dynatrace Associate Learning Path com atenção
2. **Revisar botões e subtópicos** dentro de cada seção para compreender as nuances da plataforma
3. **Completar os action items** ao final de cada lição (seção "call to action") — essenciais para reforçar o aprendizado
4. **Agendar o exame** quando se sentir confiante no entendimento do material

---

## Detalhes do Exame

| Item | Detalhe |
|---|---|
| **Questões escritas** | 60 questões (multiple-choice + multiple-response) |
| **Questões práticas** | 10 a 15 questões |
| **Multiple-response** | Pontuação parcial por acerto; mesmo percentual deduzido por erro |
| **Proctoring** | ProctorU — verbal apenas em inglês |
| **Seção prática** | Open book ✅ |

> **Atenção:** A seção prática é open book — você pode consultar documentação durante essa parte.

### Acessibilidade

Candidatos que precisam de acomodações devem criar um ticket de suporte. A Dynatrace está comprometida em garantir acessibilidade para todos os indivíduos.

---

## Pontos-Chave para a Prova

- O exame tem **60 questões escritas** + **10 a 15 questões práticas**
- A **seção prática é open book** — documentação pode ser consultada
- Questões **multiple-response** deduzem parcialmente por erro, mas **não resultam em pontuação negativa total**
- Proctoring verbal **apenas em inglês**
- Pré-requisito recomendado: **6 meses de experiência** com a plataforma Dynatrace
- Conhecimento de **cloud, containers (Docker/Kubernetes) e arquitetura SOA** é esperado
