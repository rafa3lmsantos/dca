## Monitoring & Infrastructure Observability

O Dynatrace coleta e analisa dados de **todas as camadas da stack tecnológica** — desde interações do usuário até processos de backend.

Esse fluxo contínuo de informações permite:
- Detectar anomalias
- Rastrear tendências de performance
- Identificar problemas em potencial **antes** de impactar a experiência do usuário

---

## Dynatrace Apps

Os apps do Dynatrace oferecem monitoramento e observabilidade extensivos em ambientes de TI:

- **Descoberta automática** de componentes: cloud platforms, clusters Kubernetes, bancos de dados
- **Visão unificada** de métricas, logs e traces
- **Insights com IA** para reduzir downtime e melhorar performance

São essenciais para manter experiências de alta qualidade para o usuário.

## Pontos-Chave para a Prova

- Monitoring é um elemento **fundacional** da plataforma Dynatrace
- Dynatrace coleta dados de **todas as camadas** da stack (user → backend)
- Os Apps descobrem e monitoram componentes **automaticamente**
- Integram **métricas, logs e traces** em uma visão unificada
- **OneAgent** é o agente de coleta de dados do Dynatrace
- **Kubernetes** é coberto no exame — namespaces e workloads são pontos de atenção
- **Databases app** permite análise de performance de bancos de dados
- Resultados podem ser organizados em **Notebooks**
